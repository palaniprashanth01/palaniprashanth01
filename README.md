### About

I build production AI systems — the unglamorous half of the LLM stack. Retrieval that grounds. Tools that actually work. Evals that catch the regressions nobody else does.

Right now I'm in the **Founder's Office at AICE**, shipping internal AI tooling and exploring where agentic systems break in the real world. On the side, I run [**Macha.ai**](https://machiai.netlify.app) — a code-mix writing tool that lets Indian creators write the way they actually speak: Hinglish, Tanglish, the messy beautiful middle.

I'm based in **Kanchipuram, Tamil Nadu**, hold an **M.Tech in Big Data Analytics from VIT Vellore**, and care most about LLMs, RAG, MCP, and agents you can trust in production.
